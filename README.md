# 🕹️ RetroGame

Plataforma web para la administracción alquiler y venta de consolas y videojuegos retro y modernos.

## 🚀 Enlace de Producción
El proyecto está desplegado y funcional en: https://proyecto-retrogame.onrender.com/

## 🛠️ Tecnologías Usadas
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Base de Datos: MongoDB Atlas
- Autenticación: JWT y Bcrypt

## ⚙️ Variables de Entorno
Para correr este proyecto localmente, necesitas configurar estas variables:
PORT=3000
DB_URI=mongodb+srv://<usuario>:<password>@cluster.mongodb.net/
JWT_SECRET=tu_clave_secreta
